# The-Indigo-Upright-Piano
An out-of-tune upright piano for free use in Black MIDI videos, editing, or other uses.

Contains a total of :
- 14 samples (C, F#)
- 2 versions to choose from
- infinite mod env release for keys 96+

Features to add
- Higher quality samples
- Velocity layers
